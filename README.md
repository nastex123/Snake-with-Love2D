# Snake with Love2D

Un juego de Snake estilo Dungeon Crawler desarrollado con Love2D.

## Características

- 5 etapas con 5 salas cada una (25 salas)
- Enemigos: Chaser, Patroller, Spawner y boss (teleporter)
- 12 ítems (activos y pasivos) con slots 1‑3
- Tienda con paginación 4x3
- Shaders y partículas procedurales
- Alta puntuación persistente y monedas guardadas
- Velocidad ajustable en tiempo real
- Depuración: `L` +10 monedas, `K` skip sala

## Controles

- Movimiento: `WASD` o flechas
- Aumentar/Disminuir velocidad: `+` / `-`
- Pausar/Reanudar: `Espacio` / `Esc`
- Usar ítem activo: `1`, `2`, `3`
- Depuración: `L` (añade 10 monedas), `K` (salta a siguiente sala)

## Instalación y uso

1. Instalar Love2D (versión 11.4 o superior) desde https://love2d.org/
2. Clonar el repositorio.
3. Ejecutar `love .` en la carpeta raíz del proyecto.
